---

## 🧠 Implemented Interfaces

### BaseType

```ts
export interface BaseType {
  id: number
  name: string
  price?: number
  image_url: string
  barcode_url: string
}
```

### Bowl (uses inheritance)

```ts
export interface Bowl extends BaseType {
  base_type_id?: number
  volume?: number
  slot_count: number
  shape: 'round' | 'square'
}
```

---

## 🔑 Key Concept Learned

### TypeScript Inheritance

```ts
interface Bowl extends BaseType
```

Meaning:

* Bowl automatically includes:

  * id
  * name
  * price
  * image_url
  * barcode_url
* Avoids duplication
* Makes code scalable
